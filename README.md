## 🎸 Project Overview

Implementation of a comprehensive web-based platform designed to manage the scheduling and operations of a multi-room music recording facility.
This system transitions a static frontend into a robust ASP.NET data-driven architecture to handle the full lifecycle of studio sessions.

## ⚙️ Core Features

- Client Portal: Data-bound interface for browsing studio catalogs and managing personal session history via GridView.
- Booking Engine: C# logic utilizing ExecuteScalar to perform real-time availability checks and prevent double bookings.
- Administrative Dashboard: Management interface for staff to oversee the master schedule with integrated Edit and Delete commands.
- Facility Management: Backend tools to modify studio rates, update equipment lists, and toggle maintenance modes in the SQL database.
- Data Architecture: Integration of ADO.NET for seamless communication between the web forms and Microsoft SQL Server.
- Analytics & Reporting: Automated calculation of studio utilization hours and projected revenue based on dynamic database records.
